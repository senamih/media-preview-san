# MediaPreviewSan

キャプチャーボード・Web カメラ・仮想カメラ（OBS Virtual Camera 等）の出力を、ウィンドウでリアルタイムに確認できる **軽量プレビューツール** です。録画も配信もしない「映像を見るだけ」に特化し、GPU 描画でディスプレイのリフレッシュレートに同期するため、カメラの FPS にデスクトップ全体が引きずられません。

## 特長

- **物理デバイスも仮想カメラも**：キャプチャーボードや Web カメラ（Media Foundation）に加え、OBS Virtual Camera など他アプリに見えにくい仮想カメラ（DirectShow + OpenCV）にも対応。
- **GPU 描画（Direct3D11）**：専用描画スレッドがディスプレイの V-Sync に同期。165Hz ディスプレイで 30fps のカメラを映してもデスクトップは滑らかなまま。
- **拡大時の補間を選択可能**：ニアレストネイバー（ドット感維持）／バイリニア（滑らか）／異方性。アスペクト比維持の ON/OFF も。
- **排他使用からのワンクリック復帰**：他アプリがデバイスを掴んでいるときは案内を表示。右クリック「再接続」で復帰、「接続解除」で他アプリへ制御を譲渡。
- **設定は自動保存**：デバイス・解像度・FPS・補間などを `settings.json` に保存。次回は自動接続。

## 動作環境

- Windows 10 / 11（64bit）
- .NET ランタイムのインストールは **不要**（単一 EXE に同梱した自己完結型）
- 表示したいデバイスのドライバがインストールされていること
- 仮想カメラを使う場合は、その提供元アプリ（OBS 等）で仮想カメラを開始しておくこと

## インストール

1. `MediaPreviewSan.exe` を任意のフォルダに置きます（`LICENSE` / `THIRD-PARTY-NOTICES.txt` も同じ場所に置くことを推奨）。
2. 初回起動時に設定画面が自動表示されます。デバイスと解像度を選んで OK を押すと接続が始まります。

> インストーラはありません。フォルダごとコピーすれば移動・配布できます。設定は実行ファイルと同じ場所の `settings.json` に保存されます。

## 使い方

起動すると選択中のデバイスの映像がメインウィンドウに表示されます。ウィンドウは自由に拡縮でき、位置・サイズは自動で保存されます。

ウィンドウ上で **右クリック** するとメニューが開きます。

| メニュー | 動作 |
|---|---|
| **設定** | 設定ダイアログを開きます。 |
| **再接続** | 現在の設定のまま、もう一度デバイスへ接続します。排他していた他アプリ（OBS / Teams 等）を閉じた後の復帰に使います。 |
| **接続解除** | デバイスを解放し、他アプリに制御を渡します。再び使うときは「再接続」で戻せます。 |
| **終了** | アプリを終了します。 |

### タイトルバーの見方

```
MediaPreviewSan - <デバイス名> [1920x1080 @ 60fps] 59.9fps
```

- `[1920x1080 @ 60fps]`：接続中の解像度と設定 FPS（自動画質選択時は実際に適用された公称 FPS）。
- 末尾 `59.9fps`：直近 1 秒の実測 FPS。

## 設定ダイアログ

右クリックメニューの「設定」から開きます。主な項目：

- **デバイス**：検出されたカメラ／キャプチャ機器の一覧。一覧の下に検出件数と、そのデバイスの取得・描画方式（Media Foundation か、DirectShow + OpenCV か）を表示します。
- **解像度・FPS**：先頭の「自動 - 最高画質」を選ぶと、列挙された中で最も高画質な組み合わせを自動採用します。
- **カラーフォーマット**：デバイスが複数の形式（NV12 等）を公開している場合に選択できます。
- **拡大時の補間**：ニアレストネイバー：最近傍 ／ バイリニア：双線形 ／ 異方性。
- **アスペクト比を維持**：ON で映像の縦横比を保ったままレターボックス表示。
- **再検出**：このボタンを押したときだけデバイスを実際に再列挙します（設定画面を開くだけでは再検出しません）。再検出後も、それまでの選択（デバイス・解像度・FPS・カラーフォーマット・補間・アス比）を可能な限り復元します。

設定に変更を加えて OK したときだけデバイスへ再接続し、変更があったときだけ `settings.json` に書き込みます。

## トラブルシュート

- **デバイスが一覧に出ない**
  - Web カメラ：Windows の「設定 → プライバシーとセキュリティ → カメラ」で、カメラへのアクセスと「デスクトップ アプリにカメラへのアクセスを許可する」を ON にしてください。OFF だと OS がデバイスを列挙しません。
  - 仮想カメラ（OBS 等）：提供元アプリで仮想カメラを開始してから、設定画面の「再検出」を押してください。
- **「デバイスは他のアプリで使用中です」と表示される**：別アプリ（OBS / Teams / ブラウザ等）がそのデバイスを排他使用しています。そのアプリを閉じてから、右クリックメニューの「再接続」を押してください。
- **映像が出ない／信号待ちのまま**：解像度・FPS の組み合わせがデバイス非対応の可能性があります。設定で「自動 - 最高画質」を選び直してください。
- **動作ログ**：実行ファイルと同じ場所の `MediaPreviewSan.log` に起動・接続・エラー内容が追記されます。不具合報告時に添付すると原因特定が早まります。
